Local Herbal Remedies Collection - Site Plan

Site Name

Local Herbal Remedies Collection.org

This name was selected to reflect the website's focus on collecting and showcasing local herbal remedies while attributing ownership to the developers.

Site Purpose

The website aims to preserve and promote traditional herbal knowledge by providing a platform to collect and share remedies. It will also recognize the contributions of local developers and highlight the scientific approval status and potential royalties from pharmaceuticals.

Scenarios

  • What are the benefits of using traditional herbal remedies?
  • How can I submit my own herbal remedy to the website?

Color Schema

Primary Color: Green (#4CAF50) - Used for headings and accents.

Secondary Color: Gold (#FFD700) - Used for background colors and highlights.

Typography

Font 1:Ubuntu, Arial, sans-serif - Used for headings.

Font 2:Confortaa ,Georgia, serif - Used for body text.

Project Compliance Summary

  • Naming Conventions: All files and folders are named and organized according to the course naming conventions, ensuring consistency and ease of navigation.
  • HTML Validity: All HTML is valid, semantic, contemporary, and relevant, enhancing the accessibility and structure of all pages. This includes the use of appropriate tags and attributes.
  • CSS Standards: Contemporary CSS is used, which is valid and adheres to standards, ensuring cross-browser compatibility. There are no unused or unnecessarily duplicated rules or declarations.
  • Visual Appeal: The website is visually appealing, with an exceptional focus on appropriateness for the target audience in all views (responsive) with no violations of design principles. This includes a clear and consistent layout, color scheme, and typography.
  • Relevant Content: The website does not have any placeholder or irrelevant content given the purpose and target audience. This includes accurate and meaningful copy, imagery, and meta descriptions on pages.
  • Lazy Loading: Any image or frame used below the fold utilizes native lazy loading or equivalent practice, improving page load times and performance.All images were got from SVGRepo internet site
  • HTML Form: An HTML form that meets the standards presented in the course was complete and functional, allowing users to submit their own herbal remedies with ease.
  • JavaScript Functions: More than one working function was used, demonstrating proficiency in JavaScript development.
  • DOM Interaction: DOM interaction, including selecting an element, modifying that element, and event listening with functionality, was used to create a dynamic and interactive user experience.
  • Conditional Branching: Conditional branching was used and working, ensuring the website's functionality adapts to different user inputs and scenarios.
  • Objects and Arrays: At least one object and one array were used, along with array methods, to manage and manipulate data efficiently.
  • Template Literals: Template literals were used exclusively for string manipulation and output, providing a clean and readable code structure.
  • localStorage: localStorage was used and functional, allowing data persistence across sessions and enhancing the user experience.
  • On-Time Submission: The project was submitted on time, or the student communicated any issues for not meeting the deadline far in advance of the due date, demonstrating professional responsibility.
  • Development Standards: The website is dynamic and responsive, adhering to development standards and ensuring a high-quality user experience.
